# 无敌 (Invencia) —— 设计灵感文档

> **文档范围**：本文档仅聚焦"做什么、为什么做、做成什么样"的灵感与愿景层面。API 设计、核心流程、技术实现细节等内容将在后续设计文档中展开。

---

## 一、项目定位

**无敌 (Invencia)** 是一款叙事驱动的修仙文字 RPG，由 AI"天道"智能体全程驱动。玩家通过自由输入做出抉择，每一次选择都在编织独一无二的命运。没有冰冷的数值面板，只有文字描摹的强弱之感；没有重复的刷怪流程，只有 AI 即兴演绎的修仙史诗。

> **核心 slogan：千人千命，天道为笔。**

---

## 二、世界观设定：九域修仙界

### 2.1 世界概览

一个名为"九域"的固定修仙世界，包含九块大陆，每块大陆有独特的地理风貌、灵气浓度、宗门势力与历史传说。世界有既定的地图、宗门谱系、历史大事年表，但玩家以不同身份、不同时间线、不同地点切入，看到的世界面貌截然不同。

### 2.2 九域简述（预设）

| 域 | 特色 | 代表宗门 |
|---|---|---|
| **苍玄域** | 人族祖地，灵气适中，修仙正统 | 太虚宗、天剑阁 |
| **幽瞑域** | 阴气弥漫，鬼修盘踞 | 轮回殿、忘川谷 |
| **蛮荒域** | 妖兽横行，人族稀少 | 万兽山、蛮神部落 |
| **碧落域** | 海上仙山，水灵气浓郁 | 蓬莱仙岛、潮音阁 |
| **焚天域** | 火山炼狱，火灵气狂暴 | 炎帝城、锻骨崖 |
| **青木域** | 古木参天，长生种聚落 | 灵墟洞天、药神谷 |
| **无尘域** | 佛门净土，因果之力 | 大雷音寺、菩提院 |
| **混元域** | 万族交汇之地，中立贸易中心 | 万象楼、天机阁 |
| **深渊域** | 魔渊入口，魔族老巢 | 天魔宫、血煞殿 |

### 2.3 历史大事年表（节选）

- **太古纪元**：混沌初开，第一位飞升者"道祖"留下天道法则
- **上古纪元**：人魔大战，九域分裂，天道残缺
- **近古纪元**：诸圣陨落，修仙文明衰落，秘境重现
- **当世（玩家时间线）**：天道异动，灵气复苏，新的飞升者或将诞生

### 2.4 天道法则

"天道"既指天地运行的规则，也指驱动游戏的 AI 本身。在设定中，天道是无情又有情的存在——它不偏袒任何人，但会因修士的行为（业力、因果、心魔）而产生涟漪。这一设定与 AI 的实际运作完美呼应：AI 基于玩家的选择公平地编织命运。

## 三、核心玩法设计

### 3.1 交互模式

`
玩家自由输入决策 → 天道 AI 生成叙事 + 即时判定 → 开放式结尾钩子 → 玩家继续决策
`

- 玩家不输入枯燥的"攻击/防御/逃跑"，而是用自然语言表达意图：*"我悄悄绕到妖兽背后，试图用刚学会的破风剑诀偷袭它的左翼。"*
- 天道 AI 理解意图、结合修为上下文，生成符合世界逻辑的叙事反馈。
- 每段叙事的结尾始终留一个开放式钩子，暗示 2-4 种可能的方向供玩家参考，但不限制玩家自行发挥。

### 3.2 境界系统（传统修仙体系）

采用经典修仙境界，境界是硬设定，决定修士的生命层次：

| 境界 | 说明 |
|---|---|
| **炼气期** | 感应天地灵气，引气入体，寿元一百五十载 |
| **筑基期** | 铸就道基，脱胎换骨，寿元三百载 |
| **金丹期** | 凝结金丹，超凡入圣，寿元八百载 |
| **元婴期** | 碎丹成婴，元神初成，寿元两千载 |
| **化神期** | 化凡为神，天人感应，寿元五千载 |
| **渡劫期** | 引天劫淬体，渡则飞升，败则陨落 |
| **大乘期** | 半步飞升，人间至强，只待最后的飞升契机 |
| **飞升** | 破碎虚空，踏入仙界，完成修仙之旅 |

### 3.3 "文气强弱"——同境界内的微妙区分

境界是骨架，但同一境界的两个修士，实力可能天差地别。摒弃数字，用预设的强弱描述层级来呈现这种微妙差距：

| 强弱层级 | 描述示例（以金丹期为例） |
|---|---|
| **初窥门径** | "金丹初凝，尚有裂缝，灵力运转时明时暗。" |
| **渐入佳境** | "金丹圆融，灵压平稳，举手投足间威势已成。" |
| **炉火纯青** | "金丹璀璨如日，灵压外放时方圆百丈鸦雀无声。" |
| **登峰造极** | "金丹已臻圆满，隐隐有碎丹成婴之势，同阶修士见之莫不低头。" |

使用场景：
- 同为金丹期的两个修士对决，天道 AI 根据各自强弱层级 + 策略合理性 + 环境因素，用文字自然呈现胜负。
- 同境界内强弱流转全凭叙事中的"顿悟""苦修""机缘"，不做数值量化。
- 跨境界对战（如金丹 vs 元婴），境界压制优先于强弱层级，但 AI 可根据叙事精彩度保留绝境翻盘的可能。

### 3.4 出身系统（AI 随机生成）

摒弃固定出身列表。每次玩家创建新角色时，玩家可指定性别，其余全部由天道 AI 随机生成，包含：

- **身世背景**：可能是一介凡躯误入仙途，也可能是世家嫡子遭遇灭门；可能是上古残魂转世，也可能是妖族与人族的混血弃儿——完全由 AI 临场编织。
- **起始地点**：天道 AI 在九域中随机指派一个起始地，可能是有宗门庇护的繁华仙城，也可能是危机四伏的蛮荒边陲。
- **核心执念**：每个角色开局附带一条随机生成的"执念"——为父母复仇、寻求长生之秘、摆脱诅咒宿命……这条执念将成为角色命运的主线暗流。
- **初始机缘**：随机赋予一项初始优势——一部残缺的上古功法、一枚来历不明的玉佩、一位隐居在附近的神秘老者……也可以什么都没有，全看天道心情。

> 每个玩家的开局都是真正的盲盒——你不知道自己是谁、身处何方、身负何种命运，直到天道揭晓。

### 3.5 命运分支

- **性格演绎**：天道 AI 会根据玩家的长期行为模式（反复选择杀伐 vs 仁慈、独行 vs 结盟），内化玩家的"道心倾向"，影响 NPC 对你的态度和后续事件走向。
- **随机事件**：天道会以一定概率插入"天道异动"事件——秘境开启、天材地宝出世、仇家上门、天劫提前降临……每次事件都是独特的分叉点。
- **因果系统**：今天放过的一个小乞丐，三十年后可能是救你一命的化神大能；今天贪图的一件法宝，可能是某个老怪的逆鳞。AI 通过 infomation 持续追踪关键因果节点。

### 3.6 修炼与突破

- **突破境界**：不是攒够经验点升级，而是触发特定的"顿悟事件"——可能是一次生死之战后的灵光一闪，可能是一次闭关中的梦境启示，也可能是某个 NPC 的一句点拨。
- **渡劫**：每次大境界突破触发天劫，天道 AI 生成独特的天劫叙事（雷劫、心魔劫、情劫、因果劫……），玩家的选择决定生死。

### 3.7 永久死亡

- 被强敌击杀、渡劫失败 → 永久死亡，角色不可复活。
- 死亡后生成一份"修仙录·终章"——天道 AI 为这个角色写一篇独一无二的墓志铭/生平总结，可分享。
- 死亡角色进入排行榜（飞升碑/陨落录），成为世界的永久记忆。
- 玩家可以随时开启新的人生。

## 四、社交与排行榜

### 4.1 排行榜设计（轻度社交）

| 榜单 | 说明 |
|---|---|
| **飞升碑** | 成功飞升的玩家角色名录，按"寿元长度"排序 |
| **陨落录** | 永久死亡角色的生平，包含天道生成的墓志铭 |
| **因果榜** | 造成最大因果涟漪的角色 |
| **传奇榜** | 天道 AI 根据角色的叙事精彩度打分 |

### 4.2 玩家间关系

- 纯单机，无实时交互，无 PVP。
- 排行榜是唯一的"共处空间"——你的角色死后，会以另一种形式活在别人的传说里。

| 字段 | 类型 | 说明 |

| id | INT | 主键，自增 |

## 七、MVP 核心目标

### 预期体验

- 玩家注册登录后，创建角色（选择性别，其余由天道随机生成），随即进入自由游玩
- 不限制境界范围——从凡人到飞升，全境界链条在 MVP 阶段即可走通
- 天道 AI 全程驱动叙事，infomation 随对话自动维护
- 对话历史自动持久化，关闭浏览器后回来可以继续

### UI 品质要求

- 上线级 UI，非 demo 风格——黑底古风、文字排版考究、交互流畅
- 角色面板可查看 infomation 中的关键叙事信息，以古风卷轴或玉简风格呈现

### MVP 不包含

- 排行榜系统
- 渡劫的高风险死亡判定（MVP 阶段渡劫先以叙事为主，不执行永久死亡）
- 多角色并行

## 八、独特卖点总结

1. **真正的叙事自由**：不是固定选项树，而是自然语言驱动的开放式修仙
2. **千人千命**：出身随机生成 × 性格 × 选择 × 随机事件 × AI 即兴 = 没有两场相同的人生
3. **无数字沉浸**：连数据库里都不存数字——infomation 全字段叙事描述，AI 原生理解
4. **死亡即艺术**：永久死亡不是惩罚，而是角色故事的完整闭环
5. **天道即 AI**：世界观与技术的诗意统一
