# 玄天界 —— 角色信息模板

> 版本：v2.0
> 日期：2026-05-25
> 变更：灵根体系移除。新增 combat_rating（道韵文气）。境界重设。物品/功法分级对齐。
> 用途：定义角色信息 JSON 的完整结构和字段说明。

---

## 一、设计原则

### 1.1 "标签——描述"模式

所有字段采用 `"标签——叙事描述"` 的格式。标签提供结构化的锚点，描述提供 AI 可直接理解的叙事上下文。

```
"realm": "元丹境·三重——丹田中元丹已有鸽卵大小，通体浑圆，隐隐有光泽流转..."
```

### 1.2 全叙事零数值

所有字段用画面和感受代替数字。

### 1.3 精简字段

每个维度一个概括性字段，相关信息融入描述中。

## 三、字段详解

### 3.1 顶层标量字段

#### `name`（必填，不可变）

角色名。创建时由玩家指定。格式：`"角色名——简短介绍"`。

示例：
```
"云逸——卧虎山猎户之子，独自在山中寻找失踪父亲已两年"
```

#### `gender`（必填，不可变）

创建时由玩家指定。格式：`"性别——叙事描述"`。

示例：
```
"男——十七岁少年，因常年独居山中，身上有种与年龄不符的沉默"
```

#### `appearance`（必填）

格式：`"外貌概括——叙事描述"`。包含年龄感、面容特征、身形体态、衣着、显著特征。≥ 60 字。

示例：
```
"清瘦少年——眉眼间还带着几分稚气，但下巴的线条已经开始变硬。右眉尾有一道寸许的旧疤。常年穿着打了补丁的粗布短褐，肩头挂着一张旧猎弓。"
```

#### `background`（必填）

格式：`"出身概括——叙事描述"`。包含出身地、家庭背景、成长关键事件。≥ 100 字。

示例：
```
"东荒·卧虎山猎户遗孤——父亲是卧虎山最好的猎手，两年前进山后失踪。母亲早逝。独自在山脚木屋生活，靠打猎和父亲留下的半篇吐纳口诀度日。"
```

#### `is_alive`（必填）

格式：`"存活状态——叙事描述"`。

初始值（凡人模板）：
```
"存活——刚踏上武道之路的凡人少年，对前路一无所知"
```

死亡时：
```
"陨落——于东荒·黑风岭与妖兽搏斗中力竭而亡，年仅十九岁。猎刀断为两截，至死未找到父亲。"
```

### 3.2 `combat_rating`（道韵，必填）

格式：`"道韵评级——叙事描述"`。

道韵是同一境界下区分战力的核心维度。衡量武者对天地大道的感悟深度——越接近大道本源，力量品质越高。

**评级列表**（道韵十二重，详见世界观设定）：

| 重 | 评级 | 叙事方向 |
|----|------|----------|
| 1 | 蒙昧 | 对道一无所知。力量松散。 |
| 2 | 启蒙 | 第一次触碰到"道"。力量开始收束。 |
| 3 | 初窥 | 窥见道的轮廓。力道开始有方向感。 |
| 4 | 入境 | 踏入道的门槛。力量流转有了韵律。 |
| 5 | 凝脉 | 道韵在经络中沉积。力道多了厚重的"质"。 |
| 6 | 洗髓 | 道韵涤荡全身。力量纯粹，拳重在"沉"不在"重"。 |
| 7 | 归真 | 返璞归真。力量纯然天地之力。 |
| 8 | 通明 | 道力贯通无碍。发力间隙消失。 |
| 9 | 化物 | 以道化物。一拳裹挟天地之"理"。 |
| 10 | 天人 | 举手投足暗合天地法则。 |
| 11 | 近道 | 距大道本源只差一线。力量近乎法则。 |
| 12 | 证道 | 力和道不再有别。一拳即是一道。 |

**初始值**（凡人）：
```
"蒙昧——对天地大道一无所知。每一拳打出，十分力散掉五分，力量松散如握不住的沙。战斗全凭本能，甚至连'道'这个概念都未曾听闻。"
```

**升级示例**：
```
"入境——正式踏入了道的门槛。不是学会了什么新招式，而是在一次生死搏杀之后的那个黄昏，他坐在山崖边看着落日，忽然觉得体内的力量不再像脱缰的野马——它开始有了自己的呼吸节奏。"
```

### 3.3 `cultivation`（三方境界，必填）

#### `cultivation.realm`（修为境界）

格式：`"境界·层级——叙事描述"`。

境界列表：凡人 → 炼气境 → 肉身境 → 元丹境 → 武宗境 → 武尊境 → 武王境 → 武皇境 → 武圣境 → 武帝境

**初始值**（凡人）：
```
"凡人——尚未踏入武道之路。对元气一无所知，对力量的认知停留在'能拉更硬的弓就是变强'。偶尔听游商说起'山那边有能一拳碎山的武者'，只当是醉话。"
```

**突破后示例**：
```
"炼气境·三层——引气入体三月有余。丹田中那团微弱的元气已从烛火变成了拳头大小，勉强能在经脉中运行三个周天。能感知天地间的元气了——像空气里飘着无数细小的光点。"
```

```
"元丹境·六重——元丹已有鸡蛋大小，表面光滑如镜，隐隐有淡金色泽。元力从丹中涌出时如江河奔流，远非炼气境时那涓涓细流可比。"
```

#### `cultivation.physique`（肉身境界）

格式：`"肉身境界·层级——叙事描述"`。

境界与修为共用同一体系，但独立修炼。肉身境界由淬体程度决定。

**初始值**（凡人）：
```
"凡人之躯——常年山林攀爬练就的体魄，比寻常凡人壮实。右肩旧疤、左膝逢阴雨天隐隐作痛。胸口有一处不明原因的暗伤，偶尔闷痛。"
```

**突破后示例**：
```
"肉身境·二重——元气淬体已有小成。皮肤下隐隐有一层淡金色的光泽，那是元气反复冲刷经脉留下的痕迹。一拳打在铁木上，木裂了，拳头只微微发红。"
```

#### `cultivation.soul`（元神境界）

格式：`"元神境界·层级——叙事描述"`。

元神境界由精神力、神识强度、对天地法则的感知力决定。

**初始值**（凡人）：
```
"凡人神识——精神力与常人无异。在山林中养成的警觉近乎本能，能在意识到危险前身体就做出反应。对'神识''元神'一无所知。"
```

**突破后示例**：
```
"炼气境·五重——神识初开。闭眼时能感知到十丈内的一切动静——不是听到的，是'看到'的。一只蚂蚁爬过树叶的声音，在识海中清晰如雷鸣。"
```

### 3.4 `items`（物品）

| 字段 | 说明 | 示例 |
|------|------|------|
| `name` | 物品名 | `"猎刀"` |
| `category` | 法宝\|丹药\|符箓\|材料\|兵器\|防具\|杂物 | `"兵器"` |
| `quantity` | 数量——叙事描述 | `"仅此一把——父亲留下的遗物"` |
| `grade` | 下品\|中品\|上品\|极品\|王级\|皇级\|圣级\|帝级\|神级 | `"下品"` |
| `effect` | 效果——叙事描述 | `"比寻常猎刀更长两指，刀刃上刻着歪歪扭扭的'平安'二字"` |
| `condition` | 状态——叙事描述 | `"完好——刀柄牛皮被汗水浸得发亮"` |

凡人模板要求：≥ 2 件，≤ 6 件，grade 为"下品"。

### 3.5 `skills`（功法/技能）

| 字段 | 说明 |
|------|------|
| `name` | 功法/技能名 |
| `category` | 武技\|心法\|身法\|秘术\|神通 |
| `grade` | 下品\|中品\|上品\|极品\|王级\|皇级\|圣级\|帝级\|神级 |
| `effect` | 效果——叙事描述 |
| `progress` | 修炼进度——叙事描述 |

凡人初始：`[]`（空数组）或最多 1 个下品残缺口诀片段。

### 3.6 `relationships`（人物关系）

| 字段 | 说明 |
|------|------|
| `name` | 人物名 |
| `identity` | 身份——叙事描述 |
| `description` | 关系描述——叙事描述 |

示例：
```
{
  "name": "赵大柱",
  "identity": "邻村猎户——父亲生前的老友",
  "description": "父亲失踪后唯一还会来看他的人。每月来一次，送些盐巴布料，嘴上说是顺路，其实是专门来的。是他在这世上最信任的人。"
}
```

凡人模板要求：≥ 1 人。

角色创建时使用的固定初始值：

```json
{
  "name": "（玩家指定）——凡人之躯，未知天命",
  "gender": "（玩家指定）——刚踏上武道之路的凡人",
  "appearance": "（天道生成）",
  "background": "（天道生成，与出身类型一致）",
  "is_alive": "存活——刚踏上武道之路的凡人，对前路一无所知",
  "combat_rating": "蒙昧——对天地大道一无所知。每一拳打出，十分力散掉五分，力量松散如握不住的沙。战斗全凭本能，甚至连'道'这个概念都未曾听闻。",
  "cultivation": {
    "realm": "（天道生成）——尚未踏入武道之路的凡人",
    "physique": "（天道生成）",
    "soul": "（天道生成）"
  },
  "items": [],
  "skills": [],
  "relationships": []
}
```

> `（天道生成）` 字段由角色创建 Prompt 指导 LLM 填充。`（玩家指定）` 由玩家输入。

## 五、合并规则

### 5.1 标量字段

直接覆盖：`name`, `gender`, `appearance`, `background`, `is_alive`, `combat_rating`

### 5.2 `cultivation`（三方境界）

`cultivation` 整体覆盖——如果 `realm`/`physique`/`soul` 任意一个变化，提交完整的 `cultivation` 对象。

### 5.3 数组字段

| 字段 | 匹配键 | 合并规则 |
|------|--------|----------|
| `items` | `name` | name 匹配 → 更新；不匹配 → 追加；quantity 为"无" → 删除 |
| `skills` | `name` | name 匹配 → 更新；不匹配 → 追加 |
| `relationships` | `name` | name 匹配 → 更新；不匹配 → 追加 |
