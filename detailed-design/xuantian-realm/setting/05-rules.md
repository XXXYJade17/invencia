## 世界设定

玄天界，古典东方武道世界。五块大陆——东荒、南疆、西域、北冥、中州。元气自中央中州向外逐级递减，中州最为鼎盛，东荒最为贫瘠。

五域并非人族独据：南疆为妖族祖地，北冥是魔道修士栖身之所，西域佛光普照，中州万族交汇。人族与妖族非绝对敌对，域外魔族是共同威胁。

深渊之海将东荒与其余四域隔绝——武帝可渡，之下唯靠失传法宝。边关烽火连年，封印正在松动。

阴阳、因果是底层法则。魔道是修炼路径而非邪恶标签。

## 基础设定

- **武道世界观**：古典东方，不得使用现代词汇、网络用语、西方幻想元素。
- **因果驱动**：每个选择都有后果。不经意的善意可能是未来生机的伏笔，轻率的杀伐可能是日后劫难的种子。
- **死亡**：永久死亡，角色故事的完整闭环。MVP 阶段渡劫/生死关不执行永久死亡。
- **深渊之海**：隔绝东荒与其他四域。武帝可渡，之下唯靠失传法宝。海中有一条不稳定裂缝——境界越高穿越失败率越大。魔族有特殊手段曾输送过少数魔帝通过。
- **魔族**：域外种族。与人族同修武道，境界相同。肉身天生更强，恢复力惊人。
- **魔道**：人族修炼路径。以杀证道，顺性而为，非邪恶。北冥为栖身之所。魔道修士是边关最凶悍的战士——以魔制魔。
- **神境**：武帝之上的下一个大境界。上古时代曾有众多武帝踏入神境，已超出玄天界范畴。本版本不考虑角色突破至神境，仅作为历史设定存在。
- **历史可知度**：普通人只知道近古以来的事。太古和上古的秘闻只能通过遗迹、传承、古老存在等特殊途径获取。
- **势力分级**：圣地≥武圣 | 皇朝≥武皇 | 王朝≥武王。南疆妖主由四妖帝轮流推举，妖皇境界低于妖帝。

## 境界体系

### 通用境界

武者修的是自身。不论人族、妖族还是魔族，修炼的根本是同一回事——以自身为炉鼎，将天地灵气化为己用，淬炼肉身、凝练元神、突破极限。境界之名或有不同，本质并无区别。

| 境界 | 说明 |
|------|------|
| 凡人 | 未踏入武道之路的普通人 |
| 炼气境 | 引气入体，感应天地。寿元一百五十载。分九重 |
| 肉身境 | 淬体脱胎，肉身蜕变。寿元三百载 |
| 元丹境 | 凝结元丹，力从丹生。寿元八百载。元丹即命门 |
| 武宗境 | 丹碎道立，自成一家。寿元两千载 |
| 武尊境 | 武道大成，万人尊之。寿元五千载 |
| 武王境 | 武道称王，同境无敌手。寿元万载 |
| 武皇境 | 皇者之气，一言一行皆带天地威压 |
| 武圣境 | 超凡入圣，触摸天地法则 |
| 武帝境 | 武道之极，人间至高。只差最后的证道飞升 |

> 魔族亦循此境。前三境同名，后六境人族称"武宗、武尊……"，魔族称"魔宗、魔尊……"。名异而实同。

### 三方境界独立

武者的**修为**（realm）、**肉身**（physique）、**元神**（soul）各自独立修炼，分别破境：

| 维度 | 含义 | 修炼方式 |
|------|------|----------|
| 修为 | 元力境界 | 吐纳元气、运转功法、参悟武道 |
| 肉身 | 体魄强度 | 淬体、药浴、极限锤炼 |
| 元神 | 精神/神识 | 冥想、观想、生死磨砺 |

三者按各自进度突破。三方平衡才是真正的强者。"武宗境的修为，肉身境的体魄"——元力虽强但身体承受不住，打三拳自己先气血翻涌。

### 同境界文气：道韵十二重

道韵是同一境界下区分战力的核心维度——越接近大道本源，力量品质越高。道韵与境界正交，不随境界突破自动提升，靠的是"悟"。

| 重 | 评级 | 叙事方向 |
|----|------|----------|
| 1 | 蒙昧 | 对道一无所知。力量松散，十分力散五分 |
| 2 | 启蒙 | 第一次触碰"道"。力量开始向内收束 |
| 3 | 初窥 | 窥见道的轮廓。力道有方向感 |
| 4 | 入境 | 踏入道的门槛。力量有了韵律 |
| 5 | 凝脉 | 道韵在经络沉积。力道多了厚重 |
| 6 | 洗髓 | 道韵涤荡全身。力量纯粹，重在"沉" |
| 7 | 归真 | 返璞归真。纯然天地之力 |
| 8 | 通明 | 道力贯通。发力间隙消失 |
| 9 | 化物 | 以道化物。一拳裹挟"理" |
| 10 | 天人 | 举手投足暗合天地法则 |
| 11 | 近道 | 力量近乎法则 |
| 12 | 证道 | 力和道不再有别。一拳即是一道 |

### 力量逻辑

- 境界压制是硬规则。凡人不可与武者正面对抗。
- 同一境界内，道韵层级 + 策略合理性 + 环境因素决定胜负。
- 道韵极高者可越境而战。
- 三方境界独立：实战需综合考虑修为、肉身、元神的短长。
- 妖族同境界肉身先天更优，人族在功法运用上更灵活。

### 修炼流派

武道修炼殊途同归，但理念与风格各异：

- **正道**（东荒、中州）：以武证道，循序渐进。根基扎实，后劲绵长。
- **魔道**（北冥）：以杀证道，顺性而为。非邪恶，是选择。
- **佛法**（西域）：以因果证道。不争不杀，以理服人。
- **妖族**（南疆）：侧重血脉觉醒，祖先的力量在每滴血中沉睡。
