## 概述

东荒，玄天界东方。祖脉被近古大战重创，五域之中最贫瘠的一域。下分七域：苍玄域、临渊域、落星域、云泽域、炎熄域、断崖域、荒石域。

## 势力

> 势力规则：武道圣地 ≥ 武圣坐镇，皇朝 ≥ 武皇坐镇，王朝 ≥ 武王坐镇。皇朝与王朝可视为圣地的附属势力。角色可出身于圣地、皇朝、王朝及其下属的世家、村镇、散修。已列势力之外的下属势力（如某王朝境内的武道世家、小宗门、猎户村等），可根据角色出身需要由 LLM 合理生成。

### 武道圣地

#### 无始殿

- 简介：坐落于无始秘境之中，秘境位于苍玄域东荒祖脉之上，灵气浓郁。管辖苍玄域、断崖域。
- 高阶战力：老祖（武帝）、殿主（武圣）、2位副殿主（武圣）、7位太上长老（武圣）
- 职位：老祖 → 太上长老 → 殿主 → 圣子/圣女 → 副殿主 → 长老（武皇） → 真传弟子（武皇、武王） → 内门弟子 → 外门弟子

#### 青云剑阁

- 简介：坐落于剑神谷中，位于云泽域。圣地弟子均为剑修，擅长剑法。
- 高阶战力：老祖（武圣）、阁主（武圣）、2位副阁主（武圣）、5位太上长老（武圣）
- 职位：老祖 → 太上长老 → 阁主 → 剑子/剑女 → 副阁主 → 长老（武皇） → 真传弟子（武皇、武王） → 内门弟子 → 外门弟子

#### 万宝阁

- 简介：坐落于百炼山脉腹地，位于落星域。山脉中矿藏丰富、地火旺盛，山谷中灵气浓郁，药草丰富，是东荒炼器、炼丹的天然宝地。旗下开设的万宝楼遍布东荒。管辖落星域、炎熄域。
- 高阶战力：老祖（武圣）、阁主（武圣）、1位副阁主（武圣）、3位太上长老（武圣）
- 职位：老祖 → 太上长老 → 阁主 → 圣子/圣女 → 副阁主 → 长老（武皇） → 真传弟子（武皇、武王） → 内门弟子 → 外门弟子

#### 焚炎谷

- 简介：坐落于焚天火山群深处，位于荒石域。东荒最大的火山群中。谷中武者皆喜炼体，以地火淬体，战力狂暴。
- 高阶战力：老祖（武圣）、谷主（武圣）、2位副谷主（武圣）、3位太上长老（武圣）
- 职位：老祖 → 太上长老 → 谷主 → 圣子/圣女 → 谷阁主 → 长老（武皇） → 真传弟子（武皇、武王） → 内门弟子 → 外门弟子

#### 天道宗

- 简介：坐落于东海之滨的临渊山，位于临渊域。山崖之下便是深渊之海。天道宗是东荒武道发源地，上古时代大陆东方最早踏入神境者皆出此处。近古大战中因毗邻深渊之海战场遭受波及，从此衰弱。战后重建，收徒随缘，不主动招生全凭机缘，人丁稀少。
- 高阶战力：宗主（武圣）、1位副宗主（武圣）、2位太上长老（半步武圣）、9位峰主（武皇巅峰）
- 职位：宗主 → 道子/道女 → 副宗主 → 太上长老 → 峰主 → 真传弟子（武皇、武王） → 内门弟子 → 外门弟子


### 皇朝

#### 大周皇朝

- 简介：坐落于苍玄域，东荒最强皇朝。皇室与无始殿渊源极深，皇族周氏血脉中据说蕴有上古武者的遗泽。
- 高阶战力：皇主（武皇）、2位镇国大将军（武王）、4位亲王（武王）
- 职位：皇主 → 亲王 → 镇国大将军 → 将领 → 士卒

#### 镇渊皇朝

- 简介：坐落于断崖域，扼守深渊之海沿岸。举国皆兵，世代死守深渊裂缝，不让渗透进来的魔族踏出断崖一步。镇渊之名，皇主即位时以血刻于断崖之上。
- 高阶战力：皇主（武皇）、3位镇渊大将（武王）、2位供奉（武王）
- 职位：皇主 → 供奉 → 镇渊大将 → 将领 → 士卒

#### 苍云皇朝

- 简介：坐落于云泽域，以武立国。皇族世代修剑，国内尚武成风——科举不考文章考擂台。
- 高阶战力：皇主（武皇）、2位剑帅（武王）、3位亲王（武王）
- 职位：皇主 → 剑帅 → 将领 → 士卒

#### 紫霄皇朝

- 简介：坐落于落星域，商业皇朝。皇族精通商道与炼器，国力不在疆土而在财力，军备多为万宝阁提供。落星域的陨星矿脉是紫霄皇朝最大的财富来源。
- 高阶战力：皇主（武皇）、1位供奉（武王）、2位商盟会长（武王）
- 职位：皇主 → 供奉 → 商盟会长 → 执事

#### 赤焰皇朝

- 简介：坐落于炎熄域，火山余烬之地。民风彪悍，以地火炼器闻名，出产的炎钢兵器畅销东荒。皇室赤焰氏，世代掌控地火之秘。
- 高阶战力：皇主（武皇）、2位炎将（武王）、2位亲王（武王）
- 职位：皇主 → 亲王 → 炎将 → 将领 → 士卒

#### 天武皇朝

- 简介：坐落于荒石域，东荒最好战的皇朝。荒石域寸草难生，锻造了天武皇朝全民皆兵的铁血传统。皇主之位不传嫡只传功——每一代皇主都是皇族中战力最强者。
- 高阶战力：皇主（武皇）、3位边关统帅（武王）、2位亲王（武王）
- 职位：皇主 → 亲王 → 边关统帅 → 将领 → 士卒

#### 玄阳皇朝

- 简介：坐落于临渊域，由天道宗扶持建立的皇朝，底蕴最浅。广纳散修，不问来历，境内鱼龙混杂，也因此扩张最快。
- 高阶战力：皇主（武皇）、2位大将军（武王）
- 职位：皇主 → 大将军 → 将领 → 士卒

### 王朝

王朝为皇朝的附属势力，各自统辖数郡之地。每个王朝至少有一位武王坐镇。以下列出各皇朝下属王朝：

- 大周皇朝 → 燕云王朝、朔风王朝
- 镇渊皇朝 → 铁壁王朝、寒关王朝
- 苍云皇朝 → 越岭王朝、青木王朝
- 紫霄皇朝 → 楚江王朝、赤金王朝
- 赤焰皇朝 → 熔岩王朝、灰烬王朝
- 天武皇朝 → 齐渊王朝、黑石王朝
- 玄阳皇朝 → 白霜王朝、流沙王朝
