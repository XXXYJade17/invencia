## 概述

东荒，玄天界东方。祖脉被近古大战重创，五域之中最贫瘠的一域。下分七域：苍玄域、临渊域、落星域、云泽域、炎熄域、断崖域、荒石域。

## 势力

> 势力规则：武道圣地有武圣；皇朝有武皇，王朝有武王。

### 武道圣地

#### 无始殿

- 简介：坐落于无始秘境之中，秘境位于苍玄域东荒祖脉之上，灵气浓郁。
- 高阶战力：老祖（武帝）、殿主（武圣）、2位副殿主（武圣）、7位太上长老（武圣）
- 职位：老祖 → 太上长老 → 殿主 → 圣子/圣女 → 副殿主 → 长老（武皇） → 真传弟子（武皇、武王） → 内门弟子 → 外门弟子
- 管辖领地：苍玄域、断崖域
- 附属势力：大周皇朝、镇渊皇朝

#### 青云剑阁

- 简介：坐落于剑神谷中，位于云泽域。圣地弟子均为剑修，擅长剑法。
- 高阶战力：老祖（武圣）、阁主（武圣）、2位副阁主（武圣）、5位太上长老（武圣）
- 职位：老祖 → 太上长老 → 阁主 → 剑子/剑女 → 副阁主 → 长老（武皇） → 真传弟子（武皇、武王） → 内门弟子 → 外门弟子
- 管辖领地：云泽域
- 附属势力：苍云皇朝


#### 万宝阁

- 简介：坐落于百炼山脉腹地，位于落星域。山脉中矿藏丰富、地火旺盛，山谷中灵气浓郁，药草丰富，是东荒炼器、炼丹的天然宝地。旗下开设的万宝楼遍布东荒。
- 高阶战力：老祖（武圣）、阁主（武圣）、1位副阁主（武圣）、3位太上长老（武圣）
- 职位：老祖 → 太上长老 → 阁主 → 圣子/圣女 → 副阁主 → 长老（武皇） → 真传弟子（武皇、武王） → 内门弟子 → 外门弟子
- 管辖领地：落星域、炎熄域
- 附属势力：紫霄皇朝、赤焰皇朝


#### 焚炎谷

- 简介：坐落于焚天火山群深处，位于荒石域。东荒最大的火山群中。谷中武者皆喜炼体，以地火淬体，战力狂暴。
- 高阶战力：老祖（武圣）、谷主（武圣）、2位副谷主（武圣）、3位太上长老（武圣）
- 职位：老祖 → 太上长老 → 谷主 → 圣子/圣女 → 副谷主 → 长老（武皇） → 真传弟子（武皇、武王） → 内门弟子 → 外门弟子
- 管辖领地：荒石域
- 附属势力：天武皇朝


#### 天道宗

- 简介：坐落于东海之滨的临渊山，位于临渊域。山崖之下便是深渊之海。天道宗是东荒武道发源地，上古时代大陆东方最早踏入神境者皆出此处。近古大战中因毗邻深渊之海战场遭受波及，从此衰弱。战后重建，收徒随缘，不主动招生全凭机缘，人丁稀少。
- 高阶战力：宗主（武圣）、1位副宗主（武圣）、2位太上长老（半步武圣）、9位峰主（武皇巅峰）
- 职位：宗主 → 道子/道女 → 副宗主 → 太上长老 → 峰主 → 真传弟子（武皇、武王） → 内门弟子 → 外门弟子
- 管辖领地：临渊域
- 附属势力：玄阳皇朝

### 王朝

王朝为皇朝的附属势力，各自统辖数郡之地。每个王朝至少有一位武王坐镇。以下列出各皇朝下属王朝：

- 大周皇朝 → 燕云王朝、朔风王朝
- 镇渊皇朝 → 铁壁王朝、寒关王朝
- 苍云皇朝 → 越岭王朝、青木王朝
- 紫霄皇朝 → 楚江王朝、赤金王朝
- 赤焰皇朝 → 熔岩王朝、灰烬王朝
- 天武皇朝 → 齐渊王朝、黑石王朝
- 玄阳皇朝 → 白霜王朝、流沙王朝
